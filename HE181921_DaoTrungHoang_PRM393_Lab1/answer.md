1. What is the purpose of the flutter doctor command?
- Checks whether your system is correctly set up to run Flutter.
2. What file acts as the entry point of a Flutter application?
- lib/main.dart (starts at main() function).
3. Explain the difference between Hot Reload and Hot Restart.
- Hot Reload updates UI without losing state; Hot Restart rebuilds the app and resets state.
4. How does runApp() build the widget tree?
- It sets the root widget and triggers Flutter to build the widget tree.
5. How Flutter enables cross-platform development
- Flutter uses one codebase and its own rendering engine to run on multiple platforms.
